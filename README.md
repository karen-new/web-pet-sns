# web-pet-sns


## アプリケーション概要
ペットの画像や動画を投稿できるSNSサイトです。


## 利用方法
アカウントを作成して、写真や動画と共にコメントを投稿します。プロフィールの編集や他のユーザーの投稿の検索、いいね、フォローも可能です。


## 目指した課題解決
ペットを飼っている人はペット自慢がしたいというニーズ、またかわいい、面白い動物の動画が見たいというユーザーのニーズそれぞれを満たせるよう目指しました。
